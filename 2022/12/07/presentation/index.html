环境准备

安装git

1.访问git官方地址，下载对应的安装包，进行安装，安装完毕后，点击鼠标右键，可以看到：

2.点击打开任务框，输入git version：

表示安装完毕。

安装node.js

1.访问nodejs官方地址，下载对应的安装包，进行安装。

2.安装好之后，配置环境变量，并在终端里面输入：node -v：

表示安装完毕。

安装Hexo

1.安装命令：npm install hexo -g

2.测试是否安装成功，命令：hexo -v

git配置SSH key

目的：可以免密的将本地的源码和资源上传到github，无需要每次都输账号和密码。

参考：配置和搭建参考

搭建博客

1.新建博客存放目录，输入命令：hexo init，目录下自动生成以下文件：

2.输入命令：hexo g，生成静态页面；hexo s，预览。

3.到Github上新建仓库，名称格式为“用户名.github.io”，我设置的用户名为“chaos0-0”.

4.输入命令：code .打开编辑器，找到_config.yml

5.repository仓库地址改为自己的，branch看自己的github仓库是master还是main。

6.输入命令：hexo d，即可发表到GitHub，可以在浏览器浏览了。

博客编写

主题安装

参考：主题安装

1.输入命令：npm install –save hexo-theme-fluid，然后在博客目录下创建 _config.fluid.yml，将主题的 _config.yml内容复制过去。

2.修改 Hexo 博客目录中的 _config.yml。

博客页面设置

参考：

主题的配置文档

建站参考文档

根据以上参考内容可以根据要实现的功能找到需要编辑的内容，在vscode编辑后输入：hexo g 生成静态页面，再输入：hexo d 上传到GitHub，即可修改网页内容。

文章编写

输入：hexo new “article” 之后，会在source-_posts的目录下生成对应文件，可用markdown语言编辑。

编辑完先hexo g + hexo d 即可上传到网页。
